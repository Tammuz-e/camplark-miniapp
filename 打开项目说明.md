# 微信开发者工具 - 如何正确打开项目

## 1. 打开正确的目录

**必须打开包含 `project.config.json` 的文件夹：**

```
/Users/macbook/campsite-booking
```

**不要**打开 `miniprogram` 子文件夹，否则会找不到配置。

## 2. 在微信开发者工具中

1. 点击 **「导入项目」** 或 **「+」**
2. 选择目录：`/Users/macbook/campsite-booking`
3. 如果已经导入过，检查左上角项目名称旁的路径是否正确

## 3. 若仍然看不到改动

1. 点击顶部 **「编译」** 或 **「重新编译」**（快捷键 Ctrl/Cmd + B）
2. 在模拟器中选择 **「我的」** Tab 查看 Profile 页
3. 确认底部 TabBar 的四个图标是否显示（首页、探索、收藏、我的）

## 4. 项目结构确认

```
campsite-booking/          ← 打开这个
├── project.config.json
├── miniprogram/
│   ├── app.json
│   ├── pages/
│   └── images/
│       ├── tab/      (home.png, explore.png 等)
│       └── icons/    (calendar.png, gear.png 等)
```
